6

Callouts

Callouts support four tones for different kinds of emphasis.

Info

Use info for neutral context, helpful notes, or reader orientation.

Success

Use success for approved decisions, completed milestones, or recommended paths.

Warning

Use warning for constraints, trade-offs, or things a reader should verify before acting.

Danger

Use danger for high-risk issues such as security concerns, data loss, or irreversible actions.

Details block

When should I use details?

Use details for optional background, caveats, implementation notes, or long explanations that should not distract from the main path.

Can details contain markdown?

Yes. Details bodies support bold, italic, inline code, lists, and links like Component index.

Accordion alias

Alias support

The generator treats accordion as an alias for details, so both render as expandable disclosure panels.
